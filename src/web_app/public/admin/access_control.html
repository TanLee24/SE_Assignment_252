Exception Handling

Operator tools to resolve common gate exceptions without breaking the audit trail.

Interaction flow: Select exception → system displays error state → operator may apply Manual Override (logged) if policy allows.

Offline Mode

When connectivity is lost, actions are stored locally and synced later.

Interaction flow: Offline → queue actions → back Online → Sync Queue → events applied to local sessions/occupancy and logged.
